# Weatherman

### AWS Lambda 및 OpenWeather API 기반 출근 복장 추천 알람 봇
- 0600 ~ 1800 사이 체감 온도와 강우 여부 확인
- 온도별 옷차림 도표 기준 입어야 할 옷 추천
- 우산 소지 여부 추천
- 매일 0550 슬랙으로 텍스트 화된 알람 Push
